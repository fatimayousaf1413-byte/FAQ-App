# 📚 FAQ App

A simple and interactive FAQ (Frequently Asked Questions) web application built using HTML, CSS, and JavaScript. This app allows users to expand and collapse questions in a clean accordion UI.

---

## 🚀 Features

- ✅ Accordion-style FAQ layout  
- ✅ Expand / Collapse answers  
- ✅ Smooth animations  
- ✅ Responsive design  
- ✅ Clean UI  

---

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)
