# 📚 FAQ App

A simple and interactive FAQ (Frequently Asked Questions) web application built using HTML, CSS, and JavaScript. This app allows users to expand and collapse questions to view answers in a clean accordion style UI.

---

## 🚀 Features

- ✅ Accordion-style FAQ layout  
- ✅ Expand / Collapse answers  
- ✅ Smooth animations  
- ✅ Responsive design (works on mobile & desktop)  
- ✅ Clean and simple UI  

---

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)
